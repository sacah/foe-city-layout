FoE Helper - Town overview - Copy citymap data

Currently supporting Firefox, Chrome and Edge

You need the FoE Helper browser extension installed, login to your city. Find the Town overview button on the FoE Helper toolbar. It will open a Your city modal. Press the Copy citymap data button, and paste this in the Import dialog on this site.

You can move buildings with a click + drag. Ctrl + Click on a building to delete it. Select a building from the dropdown top right and click Add to add it.

Save will store the current city layout into your Browsers storage, when you load this page again it will load this by default. Click Save again it will overwrite any previous save.
